Sound / Image, a laboratory

One sound, many ways of seeing it. Turn on the microphone and a single live signal, a voice, the room, the hall, is rendered at once through every visual dialect: lines, fields, rasters, glyphs, particles, structures. Silence reads as stillness, noise as turbulence. Press live to go full width and page through the specimens against whatever you are listening to, and decide which belong on the wall.

What this is

A live instrument, not a website. Each card is a specimen: one method for making sound visible. Press live and allow the microphone, and every specimen reads the same signal at once, the waveform, its spectrum, its loudness, its onsets. With the mic off, a gentle synthetic signal keeps the catalogue breathing so the visual register stays legible on its own.

How to read it

  • live enters a fullscreen, one-specimen-at-a-time pager; swipe or ← → to move, mic to listen.
  • Each tile opens the same way; the corner sliders icon opens its patch (parameters).
  • Filter by family or tag; search by name or lineage.
  • palette switches amber / signal (blue) / night / paper.
  • Keys: space pause, m mic, t palette, [ ] sensitivity, esc close.

How it is built

One self-contained HTML file. Every specimen is Canvas 2D against a small contract (metadata plus a draw(ctx,t,params,theme,audio) function), driven by one shared loop that runs only the visible specimens and feeds each the live audio frame. Parameters set the baseline; audio modulates around it; a per-specimen Audio drive dials the depth.
